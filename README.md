#  Awaisey Bank — ATM Machine Simulation

A simple text-based ATM simulation built with Python.  
Experience real ATM features right in your terminal!

---

##  Features

-  Secure 4-digit PIN system (3 attempts only)
-  Account gets banned after 3 wrong attempts
-  Withdraw money from your account
-  Deposit money into your account
-  Check your current balance
-  Exit anytime from the menu

---

##  How to Run

Make sure Python is installed, then:

```bash
python atm.py
```

---

##  How it Works

1. Enter your **4-digit PIN** to login
2. Choose from the menu:
   - `1` → Withdraw Money
   - `2` → Deposit Money
   - `3` → Check Balance
   - `4` → Exit
3. Wrong PIN 3 times? **Account banned!**
